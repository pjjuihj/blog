## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/pjjuihj/blog.git
cd blog

# 启动本地服务器
hugo server -D

# 访问 http://localhost:1313
```

---

## 📦 构建部署

```bash
# 构建静态文件
hugo --gc --minify

# 输出目录：./public
```

博客已配置 GitHub Pages 自动部署，推送到 `main` 分支即可触发更新。
